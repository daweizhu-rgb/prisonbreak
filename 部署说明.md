# 网页版《警察与逃犯》— 发链接分享（电脑 + 手机浏览器）

项目里已包含 **`index.html`** 网页版游戏，电脑和手机浏览器打开链接即可玩。

---

## 一、本地先试玩

1. 用浏览器直接打开 **`index.html`**（双击文件，或拖进浏览器）。
2. 电脑：用 **方向键** 移动，**空格** 跳跃。
3. 手机：用屏幕上的 **虚拟按键**（左/右/上/下/跳跃）操作。

确认本地能正常玩后，再部署到网上发链接。

---

## 二、部署到网上（发链接给他人）

任选一种方式，把 **`index.html`**（以及同目录下若有的资源）放到网站空间，然后把网址发出去即可。

### 方式 1：GitHub Pages（免费、推荐）

1. 在 [GitHub](https://github.com) 新建一个仓库（例如 `prisonbreak`）。
2. 把本地的 **`index.html`** 和 **`游戏规则.md`**（可选）上传到该仓库。
3. 打开仓库 **Settings → Pages**：
   - **Source** 选 **Deploy from a branch**
   - **Branch** 选 `main`（或你用的默认分支），文件夹选 **/ (root)**
   - 保存。
4. 几分钟后，页面会给出一个地址，形如：  
   `https://你的用户名.github.io/prisonbreak/`  
   用这个链接在电脑或手机浏览器打开即可玩。

### 方式 2：Netlify / Vercel（免费）

1. 注册 [Netlify](https://netlify.com) 或 [Vercel](https://vercel.com)。
2. 选择「从本地上传」或「从 Git 导入」：
   - 若上传：把包含 **`index.html`** 的文件夹（或整个项目）拖进网站。
   - 若导入 Git：先 push 到 GitHub，再在 Netlify/Vercel 里选该仓库并部署。
3. 部署完成后会得到一个 **https://xxx.netlify.app** 或 **https://xxx.vercel.app** 的链接，发这个链接即可在电脑和手机浏览器玩。

### 方式 3：其他静态托管

只要支持放静态网页的空间（如 Gitee Pages、自建服务器、公司内网等），把 **`index.html`** 放上去，用浏览器访问该 URL 即可。手机用同一链接即可玩。

---

## 三、注意事项

- **HTTPS**：尽量用 **https** 的链接，部分手机浏览器对 http 有限制。
- **手机操作**：网页版已带触屏虚拟键（左/右/上/下/跳跃），无需键盘。
- **单文件**：当前游戏逻辑和样式都在 **`index.html`** 里，部署时只需上传这一个文件（若你放了规则说明，可一并上传）。

把部署好的链接发给别人，对方在电脑或手机浏览器里打开即可玩。
